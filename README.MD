# Go Time Handling — Fundamentals

Time handling is one of the most important parts of backend development. This repository covers essential and practical time-related problems in Go, including:

## Covered Topics
- Getting current time
- Formatting time using layouts
- Parsing custom date formats
- Working with durations
- Add / subtract time
- Unix timestamps
- UTC vs Local time
- Basic token expiration check

This repo is meant for developers who want a strong foundation in Go’s time package before moving into advanced topics like timers, rate limiting, or distributed clock handling.
